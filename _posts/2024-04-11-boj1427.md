### #문제
[백준 1427번 소트인사이드](https://www.acmicpc.net/problem/1427)

### #풀이
```c++
#include <algorithm>
#include <iostream>
#include <string>

using namespace std;

int main()
{
	ios::sync_with_stdio(false); cin.tie(NULL);

	int N;
	cin >> N;
	string S = to_string(N);

	sort(S.rbegin(), S.rend());

	for (int i = 0; i < S.size(); ++i)
	{
		cout << S[i];
	}

	return 0;
}
```

### #정리
입력 받은 숫자를 string으로 변환 후, 역으로 정렬하여 출력.
